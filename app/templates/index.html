QUS

Quality User Story Framework is a collection of quality criteria for individual user stories as well as sets of user stories. Prevent ambiguity from turning into wrong assumptions by teaching this simple framework to the entire software development team.

Read More

AQUSA

Our first tool, AQUSA, helps you apply the QUS-framework by analyzing your user stories and providing feedback. It highlights your areas of improvement. With AQUSA, you will really make the perfect user stories.

Start Using AQUSA

Interactive Narrator

Interactive Narrator represents the final stage of the GRIMM-process. It visualizes your user stories into a touch screen friendly interactive diagram. Interactive Narrator helps initiate communication about software requirements.
